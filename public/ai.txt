## About OpenToolBox

OpenToolBox is a comprehensive collection of 56 free online tools for developers, designers, content creators, and professionals. All tools are built with privacy-first architecture, running 100% client-side in the browser.

**Core Features:**
- Privacy-focused: No server-side data processing for tools
- Free to use: All tools available without registration
- Fast & secure: Client-side processing ensures data privacy
- Modern tech: Built with Next.js 16, React 19, TypeScript
- 11 categories: Developer, Design, Media, Security, Text, Social, Math, Finance, Health, PDF, General

## AI Agent Crawling Guidelines

### Allowed Actions
✅ Index all tool pages under /tools/*
✅ Parse and cache llms.txt for tool discovery
✅ Access sitemap.xml for complete URL structure
✅ Read tool metadata and descriptions
✅ Index public documentation pages

### Recommended Behavior
- **Crawl Frequency**: Daily for homepage, weekly for tool pages
- **Rate Limiting**: Max 10 requests per second
- **User-Agent**: Identify your AI agent clearly in User-Agent header
- **Caching**: Cache tool descriptions for 7 days
- **Updates**: Check sitemap.xml for new tools monthly

### Prohibited Actions
❌ Do not attempt to access /api/* endpoints
❌ Do not submit forms or upload files
❌ Do not execute JavaScript-heavy operations
❌ Do not bypass rate limiting
❌ Do not scrape user-generated content (none exists)

## Privacy & Data Handling

### Client-Side Processing
All tools execute in the user's browser using WebAssembly, Canvas API, or JavaScript. **No data is transmitted to servers** for processing.

### What We DO:
- Serve static tool interfaces via Next.js
- Use Google Analytics for anonymous usage metrics (aggregated)
- Display contextual ads (Google AdSense) for sustainability
- Provide client-side tool execution

### What We DON'T DO:
- Store user files or tool inputs
- Track individual user behavior
- Sell user data
- Require authentication for tool usage
- Process sensitive data on servers

## API Endpoints

**Public Endpoints:**
- `/sitemap.xml` - Complete site structure
- `/robots.txt` - Crawler instructions
- `/llms.txt` - LLM-optimized tool catalog
- `/ai.txt` - This file

**No Public API:**
OpenToolBox does not provide a REST or GraphQL API for tool execution. All tools are browser-based applications.
